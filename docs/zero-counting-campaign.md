# 零点計数キャンペーン設計 — 「区間内に零点がある」から「最初の零点である」へ

背景: PROMPT/2 の注意点。v0.1 で kernel-checked になったのは
「[14.1, 14.2] に臨界線上零点が少なくとも一つ存在する」であり、
「第1零点」と正式に言うには N(14.1) = 0, N(14.2) = 1 相当が必要。

## 目標 (Lean で述べる形; N(T) 関数の定義は避け内容を直接述べる)

- **Z-A (低高さ零点フリー)**:
  `∀ s : ℂ, 0 ≤ s.re → s.re ≤ 1 → 0 < s.im → s.im ≤ 141/10 → riemannZeta s ≠ 0`
- **Z-B (区間内一意)**:
  `∃! t : ℝ, 141/10 ≤ t ∧ t ≤ 71/5 ∧ riemannZeta (1/2 + t·I) = 0`
  (加えて帯 σ ≠ 1/2, t ∈ [14.1, 14.2] の零点フリー)
- (単純零点性は別スライス: ζ'(ρ) ≠ 0 相当。Z-B の Ξ' ≠ 0 から臨界線方向の
  単純性は従うが、複素方向の単純性主張は追加検討)

## mathlib 調査結果 (v4.32)

- **偏角原理・Rouché・巻き数による零点計数は未収録**
  (Analysis/Analytic/IsolatedZeros, Uniqueness まで)。経路Aは原理の形式化
  自体が大規模プロジェクトになる。
- 関数等式は収録済み: `riemannZeta_one_sub`, `completedRiemannZeta_one_sub`,
  共役対称は本ラボの zeta-conj [既促] / lambda0-conj が既にある。

## 経路A: 偏角原理の形式化 (非推奨・保留)

∮ ζ'/ζ による計数。必要部品: 閉曲線積分・偏角の連続追跡・零点の重複度理論。
mathlib への一般的貢献価値は高いが、見積り数週間〜数ヶ月規模。本ラボの
「数値証明書 + 初等解析」路線から外れる。保留。

## 経路B (推奨): 被覆 + 対称性 + 単調性 — 全て既存機構の延長

### 数学的骨格

1. **対称性による削減**: ζ の非自明零点は ρ ↔ ρ̄ (実係数) と ρ ↔ 1−ρ
   (関数等式) で対称。よって σ ∈ [1/2, 1] × (0, 14.1] の零点フリーを示せば
   帯全体 0 ≤ σ ≤ 1 で零点フリー (境界 σ=1 は既収録の
   completed-zeta-ne-zero-of-one-le-re 系で処理)。
2. **η への転送**: ζ(s) = η(s)/(1−2^{1−s})。除数の零点は σ=1 直線上
   (s = 1 + 2πik/ln2) のみで対象領域内部に無し → 「η ≠ 0 on 領域」を示せば良い。
3. **被覆論法**: コンパクト長方形 R = [1/2, 1] × [0, 14.1] を格子点 {s_j} で
   被覆し、各点で ‖η(s_j)‖ ≥ m_j > 0 の球証明書 + 大域 Lipschitz 定数
   ‖η'‖ ≤ M on R から、格子間隔 δ_j < m_j/M で R 全体の η ≠ 0 を得る。
   - 既存部品: η 値球 (m=4 Boole、任意点で半径 ~0.01) ✓
   - 新規部品 **Z1: η' の Boole 型評価** — η'(s) = Σ(−1)^n (−ln n) n^{−s} の
     誤差定理。既存の cpow-diff 梯子と同じ骨格に ln n 重みが付くだけ
     (ln n 球 ✓ 既存、(ln n)·n^{−s} 球 = ball-mul ✓ 既存)。
     粗い大域上界 M (例: ‖η'‖ ≤ 15 on R) で足りるので m=2 程度で十分。
4. **点数見積り**: |η| は R の大部分で O(1)、最小は t=14.1 付近 σ=1/2 近傍の
   ~0.066。適応格子 (|η| に比例した δ) で **~2,000〜5,000 点**。
   チャンク化 (~100 点/claim) で 20〜50 claims、計算 2〜6 時間 —
   certify-eta-partial の流用 + 新コンパイラ certify-eta-nonzero-grid。
5. **帯 [14.1, 14.2] の一意性**:
   - 線外: 同じ被覆を帯 σ ∈ (1/2, 1] × [14.1, 14.2] に適用 (小さい追加)。
   - 線上: **Z4: Ξ' < 0 on [14.1, 14.2]** (真に単調 → 交差はちょうど1回)。
     Ξ'(t) = re(i·ξ'(1/2+it)) を因子の導関数球で評価:
     ζ' は Z1 の η' から、π^{−s/2} の導関数は log π 球 ✓ 既存、
     **Γ'(z) が主要新規部品** — 候補: (i) Kummer 恒等式の項別微分
     (T_n の z-微分は有理式 → 厳密有理計算のまま; e^{−X}X^z 部の微分は
     log X 球 ✓)、(ii) 差分商 + Γ'' 粗上界。(i) が本命 — 新規数学は
     partialGamma の z-微分の可換性 (積分記号下の微分、mathlib の
     hasDerivAt_integral_of_dominated_loc_of_lip 系で形式化可能) のみ。
6. **t ∈ [0, ε] の処理**: t=0 近傍は |η| が大きく (η(σ) ≥ 0.6 for σ≥1/2)、
   被覆が粗くて済む。特別扱い不要の見込み。

### スライス分割

- **Z1**: η' 誤差定理 (m=2 Boole、ln-重み) + 大域 ‖η'‖ ≤ M claim。1〜2ループ。
- **Z2**: certify-eta-nonzero-grid コンパイラ (適応格子生成 + ‖η(s_j)‖ ≥ m_j
  球 fan-out + Lipschitz 結合 claim の機械生成)。2〜3ループ + 計算数時間。
- **Z3**: 対称性の橋渡し claims (関数等式・共役で領域削減を正当化)。1ループ。
- **Z4**: Γ' Kummer 微分 + Ξ' < 0 on [14.1,14.2]。3〜5ループ (最難)。
- **Z5**: Z-A / Z-B の最終合成。1ループ。

総見積り: **8〜12 ループ + 計算 4〜8 時間**。新規数学は Z1 (軽) と Z4 の
積分記号下微分 (中) のみで、残りは実績ある機構の再配線。

## リスク

- 被覆点数が見積りを超える場合 (|η| の予想外の小ささ): 適応格子の再分割で
  対応可能だが計算時間が線形に伸びる。
- Γ' の積分記号下微分の mathlib 形式化が想定より重い場合: 差分商 + Γ'' 上界の
  経路 (ii) に切替 (Γ'' は Kummer 級数の2階差分で粗く抑えられる)。
- Ξ' の符号余裕: Ξ'(t₀) ≈ −1.4e-4/0.1 スケール — Ξ 値球の精度 (~1.8e-5) と
  同程度の余裕が必要。導関数球の精度目標は ~3e-5/0.1 = 3e-4 で足りる見込み。
