# 零点計数キャンペーン設計 — 「区間内に零点がある」から「最初の零点である」へ

背景: PROMPT/2 の注意点。v0.1 で kernel-checked になったのは
「[14.1, 14.2] に臨界線上零点が少なくとも一つ存在する」であり、
「第1零点」と正式に言うには N(14.1) = 0, N(14.2) = 1 相当が必要。

## 目標 (Lean で述べる形; N(T) 関数の定義は避け内容を直接述べる)

- **Z-A (低高さ零点フリー)**:
  `∀ s : ℂ, 0 ≤ s.re → s.re ≤ 1 → 0 < s.im → s.im ≤ 141/10 → riemannZeta s ≠ 0`
- **Z-B (区間内一意)**:
  `∃! t : ℝ, 141/10 ≤ t ∧ t ≤ 71/5 ∧ riemannZeta (1/2 + t·I) = 0`
  (加えて帯 σ ≠ 1/2, t ∈ [14.1, 14.2] の零点フリー)
- (単純零点性は別スライス: ζ'(ρ) ≠ 0 相当。Z-B の Ξ' ≠ 0 から臨界線方向の
  単純性は従うが、複素方向の単純性主張は追加検討)

## mathlib 調査結果 (v4.32)

- **偏角原理・Rouché・巻き数による零点計数は未収録**
  (Analysis/Analytic/IsolatedZeros, Uniqueness まで)。経路Aは原理の形式化
  自体が大規模プロジェクトになる。
- 関数等式は収録済み: `riemannZeta_one_sub`, `completedRiemannZeta_one_sub`,
  共役対称は本ラボの zeta-conj [既促] / lambda0-conj が既にある。

## 経路A: 偏角原理の形式化 (非推奨・保留)

∮ ζ'/ζ による計数。必要部品: 閉曲線積分・偏角の連続追跡・零点の重複度理論。
mathlib への一般的貢献価値は高いが、見積り数週間〜数ヶ月規模。本ラボの
「数値証明書 + 初等解析」路線から外れる。保留。

## 経路B (推奨): 被覆 + 対称性 + 単調性 — 全て既存機構の延長

### 数学的骨格

1. **対称性による削減**: ζ の非自明零点は ρ ↔ ρ̄ (実係数) と ρ ↔ 1−ρ
   (関数等式) で対称。よって σ ∈ [1/2, 1] × (0, 14.1] の零点フリーを示せば
   帯全体 0 ≤ σ ≤ 1 で零点フリー (境界 σ=1 は既収録の
   completed-zeta-ne-zero-of-one-le-re 系で処理)。
2. **η への転送**: ζ(s) = η(s)/(1−2^{1−s})。除数の零点は σ=1 直線上
   (s = 1 + 2πik/ln2) のみで対象領域内部に無し → 「η ≠ 0 on 領域」を示せば良い。
3. **被覆論法**: コンパクト長方形 R = [1/2, 1] × [0, 14.1] を格子点 {s_j} で
   被覆し、各点で ‖η(s_j)‖ ≥ m_j > 0 の球証明書 + 大域 Lipschitz 定数
   ‖η'‖ ≤ M on R から、格子間隔 δ_j < m_j/M で R 全体の η ≠ 0 を得る。
   - 既存部品: η 値球 (m=4 Boole、任意点で半径 ~0.01) ✓
   - 新規部品 **Z1: η' の Boole 型評価** — η'(s) = Σ(−1)^n (−ln n) n^{−s} の
     誤差定理。既存の cpow-diff 梯子と同じ骨格に ln n 重みが付くだけ
     (ln n 球 ✓ 既存、(ln n)·n^{−s} 球 = ball-mul ✓ 既存)。
     粗い大域上界 M (例: ‖η'‖ ≤ 15 on R) で足りるので m=2 程度で十分。
4. **点数見積り**: |η| は R の大部分で O(1)、最小は t=14.1 付近 σ=1/2 近傍の
   ~0.066。適応格子 (|η| に比例した δ) で **~2,000〜5,000 点**。
   チャンク化 (~100 点/claim) で 20〜50 claims、計算 2〜6 時間 —
   certify-eta-partial の流用 + 新コンパイラ certify-eta-nonzero-grid。
5. **帯 [14.1, 14.2] の一意性**:
   - 線外: 同じ被覆を帯 σ ∈ (1/2, 1] × [14.1, 14.2] に適用 (小さい追加)。
   - 線上: **Z4: Ξ' < 0 on [14.1, 14.2]** (真に単調 → 交差はちょうど1回)。
     Ξ'(t) = re(i·ξ'(1/2+it)) を因子の導関数球で評価:
     ζ' は Z1 の η' から、π^{−s/2} の導関数は log π 球 ✓ 既存、
     **Γ'(z) が主要新規部品** — 候補: (i) Kummer 恒等式の項別微分
     (T_n の z-微分は有理式 → 厳密有理計算のまま; e^{−X}X^z 部の微分は
     log X 球 ✓)、(ii) 差分商 + Γ'' 粗上界。(i) が本命 — 新規数学は
     partialGamma の z-微分の可換性 (積分記号下の微分、mathlib の
     hasDerivAt_integral_of_dominated_loc_of_lip 系で形式化可能) のみ。
6. **t ∈ [0, ε] の処理**: t=0 近傍は |η| が大きく (η(σ) ≥ 0.6 for σ≥1/2)、
   被覆が粗くて済む。特別扱い不要の見込み。

### スライス分割

- **Z1**: η' 誤差定理 (m=2 Boole、ln-重み) + 大域 ‖η'‖ ≤ M claim。1〜2ループ。
- **Z2**: certify-eta-nonzero-grid コンパイラ (適応格子生成 + ‖η(s_j)‖ ≥ m_j
  球 fan-out + Lipschitz 結合 claim の機械生成)。2〜3ループ + 計算数時間。
- **Z3**: 対称性の橋渡し claims (関数等式・共役で領域削減を正当化)。1ループ。
- **Z4**: Γ' Kummer 微分 + Ξ' < 0 on [14.1,14.2]。3〜5ループ (最難)。
- **Z5**: Z-A / Z-B の最終合成。1ループ。

総見積り: **8〜12 ループ + 計算 4〜8 時間**。新規数学は Z1 (軽) と Z4 の
積分記号下微分 (中) のみで、残りは実績ある機構の再配線。

## リスク

- 被覆点数が見積りを超える場合 (|η| の予想外の小ささ): 適応格子の再分割で
  対応可能だが計算時間が線形に伸びる。
- Γ' の積分記号下微分の mathlib 形式化が想定より重い場合: 差分商 + Γ'' 上界の
  経路 (ii) に切替 (Γ'' は Kummer 級数の2階差分で粗く抑えられる)。
- Ξ' の符号余裕: Ξ'(t₀) ≈ −1.4e-4/0.1 スケール — Ξ 値球の精度 (~1.8e-5) と
  同程度の余裕が必要。導関数球の精度目標は ~3e-5/0.1 = 3e-4 で足りる見込み。


## Z2 実装確定設計 (第50ループ)

### 実測ベースの修正見積り

中域 t ∈ [8,13] で N=15 の m=4 誤差が 0.1 超となり不足 (設計時見積りの誤り)。
N 段階を {8 (t≤3), 15 (t≤6.5), 22 (t≤9.5), 30 (t≤14.1)} に修正した結果、
セル数 ~13,000・総計算 **15〜20時間** (当初 2〜6h の 2.5〜3 倍)。
数学的障害はなく純粋な計算量。再開可能なチャンク設計で続行する。

### claim アーキテクチャ (確定)

1. **rotor** (n, δ ごと): 既存 certify-cpow を a=0 で流用 —
   ‖n^{−δi} − (C−S·I)‖ ≤ ~2e-7。検証済み (rotor-test-n2)。
2. **u-chain** (n, 列共通 t-格子, ~20 ステップ/claim):
   n^{−i t_{j+1}} = n^{−i t_j} · n^{−iδ} (cpow_add) + ball-mul + 再中心化。
   回転はノルム保存なので半径は線形成長 (~2e-7/step)。
3. **grid-block** (列 × 行チャンク, ~6 セル/claim, heartbeat 上限 4M に引上げ):
   各セル: (i) アンカー W̃(s₀) 球 = Σ±n^{−σc}·u_{n,j} (n^{−σc} は
   norm_num 冪比較で無料)、(ii) eta-boole4-uniform [2c18454eb321] で誤差、
   (iii) dirichlet-poly-lipschitz [b01e70c02524] + boole4-boundary-lipschitz
   [0c32da8883ce] (m は列の左端 σ に適応 — 右半分で δ 倍増)、
   (iv) nonzero-of-ball-chain [5df10af27204] で η ≠ 0。
   結論: ∀ s ∈ セル, RH.dirichletEtaEntire s ≠ 0 の連言。
4. **coeff** (N, m ごと): Σ ln n·n^{−m} ≤ ML 等 (auto-log-n 球 + norm_num)。
5. **領域統合**: ブロック連言 → 列 → 領域 → σ∈[1/2,1]×(0,14.1] 全域
   (小さい合成 claims)。

精度: 球 den 1e6 (マージン ~0.05-0.5 に対し十分粗くて良い)。

## 設計改訂 (第53ループ): 被覆の2つの原理的障害と解決

### 障害1: η の除数零点 s₀ = 1 + 2πi/ln2 ≈ 1 + 9.0647i

η(s) = (1−2^{1−s})ζ(s) は s₀ で消える (ζ(s₀) ≠ 0 だが前因子が零)。
σ = 1 付近 × t ≈ 9.06 の小矩形は η 被覆では覆えない。

**解決 (Z2s)**: 例外矩形 B = [1−μ, 1]×[t₀±ν] を λ₃(s) = (1−3^{1−s})ζ(s) で被覆。
- |λ₃| ≈ 2.5 on B (巨大マージン; d₃ の零点 t = 2πk/ln3 = 5.72k は B から遠い)
- 3項グループ化 g_k = (3k+1)^{−s} + (3k+2)^{−s} − 2(3k+3)^{−s} は絶対収束
  (‖g_k‖ ≤ 3‖s‖(3k)^{−σ−1}; cpow-sub-cpow-norm-bound [b3eea90b06d1] +
  rpow-tail-sum-bound [b4fe6ed2befb] で裾処理、Boole 加速不要)
- 収束→解析接続の骨格は η で実証済みのパターンを流用
  (79fb7b4b8ccb: CauchySeq → TendstoLocallyUniformlyOn → 恒等定理)
- 転送: (1−3^{1−s})ζ ≠ 0 → ζ ≠ 0 は mul_ne_zero で無料
- σ ≥ 1 は mathlib riemannZeta_ne_zero_of_one_le_re (s=1 含む) で無料

### 障害2 (より深刻): 線外零点排除は被覆では原理的に不可能

Z-B の「帯 [14.1,14.2] × σ≠1/2 零点フリー」被覆案は成立しない:
ζ の零点 ρ ≈ 1/2 + 14.1347i の近傍では |η| ~ 1.2·|s−ρ| → 0 なので、
σ → 1/2⁺ に近づく線外領域のセルは無限に細かくなる (被覆爆発)。
同様に Z-A の角 (σ≈1/2, t→14.1) もセル数 ~10⁴ に爆発する。

**解決 (Z6: 3零点ディスク論法)** — 偏角原理を回避する初等的計数:
f = ξ (completed zeta; 実軸対称 + s↔1−s 対称) について、中心 c = 1/2 + i·t_c
(t_c ≈ 14.05)、半径 r (≈0.3) と R (≈1.7–2.0) の同心円で:

  補題 (3零点上界): f が closedBall(c,R) で解析的 (DiffContOnCl)、
  z₁,z₂,z₃ ∈ closedBall(c,r) が相異なる零点、sphere(c,R) 上 ‖f‖ ≤ M なら
    ‖f(c)‖ ≤ M · r³/(R−r)³
  証明: g = f/((·−z₁)(·−z₂)(·−z₃)) は除去可能特異点定理で球上解析的、
  境界で ‖g‖ ≤ M/(R−r)³、最大値原理 (mathlib AbsMax) で ‖g(c)‖ を抑え、
  ‖f(c)‖ = ‖g(c)‖·Π‖c−zᵢ‖ ≤ M·r³/(R−r)³。Blaschke 因子不要の粗い版で足りる。

  適用: ‖ξ(c)‖ > M·r³/(R−r)³ を数値証明書で示せば D(c,r) 内の相異なる零点は
  高々 2 個。線外零点 β+it′ は関数等式で対 (1−β)+it′ を持ち (c が臨界線上
  なので同距離 → 両方 D 内)、既証明の線上零点 (符号変化 [07abd9b5cfa2]) と
  合わせて 3 個の相異なる零点 → 矛盾。よって **D(c,r) 内の零点はすべて線上**。
  線上の一意性は Ξ′ < 0 (Z4) が与える。

  必要部品:
  - 除去可能特異点による因子分解 ×3 (mathlib RemovableSingularity) — 新規 2–3 claims
  - 最大値原理適用 (mathlib norm_le_of_forall_mem_frontier系) — 新規 1–2 claims
  - sphere(c,R) 上の ‖ξ‖ 上界: 弧分割 (~24–32 弧) × [π冪 · Γ(s/2)球 · ζ球(η機構) ·
    |s(s−1)/2|] — Γ の弧点 Kummer 連鎖 ~24–32 点 (Z4 の Γ′ 機構で弧内 Lipschitz 補間)
  - ‖ξ(c)‖ 下界: 既存機構 (Γ(1/4+it_c/2·i) Kummer + η 球 + 除数上界)
  - ξ の対称性で円の σ<1/2 側の M 評価は σ>1/2 側に帰着 (arg対称)

  数値目安 (f64 概算; 計画器で厳密化): ‖ξ(c)‖ ≈ 1.2e-4, M(R=2.0) ≈ 1.2e-2,
  (r/(R−r))³ = (0.3/1.7)³ ≈ 5.5e-3 → 上界 ≈ 6.6e-5 < 1.2e-4 (×1.8 マージン)。

### 改訂後の全体構成

- Z-A: mathlib(σ≥1) ∪ η被覆(σ∈[1/2,1], t≤14.1, D外, B外) ∪ λ₃被覆(B) ∪
  ディスクD内は3零点補題+線上符号変化位置で t<14.1 零点なし ∪ 対称性(σ<1/2)
- Z-B: D ⊇ 帯[14.1,14.2]×[0.2,0.8]-近傍: 線外ゼロなし (3零点補題)、
  線上一意 (Ξ′<0, Z4)、σ∈[0.8,1]×帯 は η被覆の小追加
- 被覆規模: 角爆発が消え ~6,000 セル (D境界 |η| ≥ 0.26)、
  チャンク+並列で ~10–15h 見込み

## 経路C (第62ループ, 数値実証済み): ディスク列 + 1次元Ξ被覆 — 最終形

mpmath による end-to-end 数値検証で、線外零点排除がディスク論法で閉じることを確認:
- Z-B ディスク c=1/2+14.05i (r=0.09,R=0.7): 3零点マージン 22.9×, 2零点 3.37×
- Z-B ディスク c=1/2+14.15i (r=0.08,R=1.2): 3零点 18.5×, 2零点 1.32×
- **Z-A ディスク列** (r=0.12, R=0.7, 中心 t=0.4..14.0 を 0.2 刻み): 全中心で
  2零点マージン >1、worst = 2.57× (t≈14.0)。臨界線上 |ξ| は t=14.1 で 4.9e-5 まで
  減るが正で、下界被覆可能。

### 再構成された Z-A/Z-B (経路C)

**Z-A (0<t≤14.1, 全帯 0≤σ≤1 で ζ≠0)**:
1. **線上**: Ξ(t)=ξ(1/2+it)>0 の1次元被覆 [0,14.1] (ξは実、cheap)。線上零点なし。
2. **線近傍 σ∈[0.38,0.62]**: ディスク列 (r=0.12, ~70中心)、2零点bound。
   線外零点は関数等式で対 (β,γ)↔(1−β,γ) → 同高2零点 → margin>2.5 で矛盾。
3. **σ∈[0.62,1]**: η/λ₃ 被覆 (|η|良好、モンスター解消 — 左端0.62で near-line爆発回避)。
4. **σ<0.5**: 関数等式 (reflect-pair)。 **σ≥1**: mathlib。

**Z-B ([14.1,14.2] 一意)**:
1. **線上**: Ξ′<0 (Z4) → 狭義単調 → ≤1零点; Ξ(14.1)>0,Ξ(14.2)<0 → ちょうど1。
2. **線外 (帯内)**: ディスク (線上既知零点14.1347 + 線外対 = 3零点、margin>4)。
