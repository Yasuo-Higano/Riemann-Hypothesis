## 方式比較

### 案A: Spouge 近似 (推奨)

Γ(z+1) = (z+a)^{z+1/2} e^{−(z+a)} (2π)^{1/2} [c₀ + Σ_{k=1}^{a−1} c_k/(z+k) + ε_a(z)]

- 係数: c₀ = 1, c_k = (−1)^{k−1}/(k−1)! · (a−k)^{k−1/2} e^{a−k}
  — **√ と e 冪のみ**: 既製の球部品 (exp 球・norm-le/ge-of-normsq・
  posreal-cpow) で全て証明書化できる。
- 誤差: re z > 0 で |ε_a(z)| ≤ a^{−1/2} (2π)^{−(a+1/2)}
  — a に対し一様、**z 依存なし**が最大の利点 (t=7 でも劣化しない)。
  a = 10 で ~2.6e-9、a = 5 で ~1.5e-5。
- 必要な新形式化: Spouge の誤差定理そのもの。原論文 (Spouge 1994, SIAM J.
  Numer. Anal. 31) の証明は Binet/Cauchy 積分表示経由 — mathlib の
  Complex.Gamma 積分表示 (GammaIntegral, re z > 0) から出発できる。
  スライス分割の見立て:
  - G1: mathlib Γ 積分表示の復習と部分積分補題 (Γ(z+1) = zΓ(z) は既製)
  - G2: Spouge 核の被積分関数分解 (有限和 + 剰余積分)
  - G3: 剰余積分の絶対値評価 (a^{−1/2}(2π)^{−(a+1/2)})
  - G4: 球インスタンス化 (係数 c_k 球 fan-out + (z+a)^{z+1/2} 球 +
    有限和の複素球合成 — 全て既製コンパイラ)
  G1–G3 が数学本体 (Abel/Boole 級の難度、2〜4 ループ想定)。

### 案B: Stirling + Binet 剰余

log Γ(z) = (z−1/2)log z − z + (1/2)log 2π + J(z)、|J(z)| ≤ 1/(8·re z·cos²(θ/2))
- 誤差が z 依存 (re z = 1/4 は小さく分母が痛い)。re を上げるには
  Γ(z) = Γ(z+n)/((z)(z+1)⋯(z+n−1)) の再帰シフト (逆数球 ✓ 複素積球 ✓) を
  併用 — シフト自体は既製部品で機械的。
- Binet 第2公式の剰余評価の形式化が本体。案Aと難度同等か。

— polygamma 系列の評価が必要で発散気味。**不採用**。

## 経路決定 (第37ループ, 2026-07-19)

mathlib API 調査の結果:
- `Complex.Gamma_eq_integral` (re z > 0 で Γ = ∫_{Ioi 0} e^{−t} t^{z−1}) ✓ 存在
- `Complex.GammaSeq_tendsto_Gamma` (Euler 極限) ✓ 存在するが**収束レートなし**
- `Stirling.tendsto_stirlingSeq_sqrt_pi` (実 Stirling) ✓ 存在するが**レートなし**

**決定: 案D「積分表示の直接区分求積」を主経路に採用**(案A Spouge・案B Binet
公式は降格)。理由: Spouge/Binet の誤差定理は文献の複素解析 (Cauchy 積分表示)
の新規形式化が必要な「未知の壁」だが、案Dは全評価が自前 MVT/Taylor 有界化
(実績のある手法) に還元され、未形式化定理への依存がゼロ。
